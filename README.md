# myslider
这是一款轻量级的jquery轮播图插件
1.动画类型
 -淡入淡出效果
 -左右滑动效果
 -自动轮播效果
2.调用方法
  $("element").mySlider();
3.相关配置
  animate:     动画效果选择   默认fadeIn
  circleEvent: 圆圈按钮事件类型  默认mouseover,
  arrowsBtn:   是否需要左右箭头按钮   默认true,
  circleBtn:   是否需要圆圈按钮，默认 true, 
  autoSlide:   是否自动轮播，默认false
  speed:       轮播速度，默认1500ms
  autoSlideSpeed:  自动轮播速度。默认2000毫秒
